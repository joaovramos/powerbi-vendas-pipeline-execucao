# Vendas — Pipeline e Execução (Power BI)

## Objetivo
Consolidar dados de vendas e pipeline comercial para análise de
conversão, desempenho por etapa, ticket médio e realização
versus metas.

## Contexto do Projeto
Projeto demonstrativo utilizando dados sintéticos estruturados
para simular operações comerciais reais.

## Escopo Analítico
- Pipeline por estágio
- Conversão por etapa
- Vendas realizadas
- Metas comerciais

## Principais Indicadores
- Taxa de conversão
- Ticket médio
- Receita realizada
- Meta vs realizado
- Tempo médio de ciclo de vendas

## Ferramentas Utilizadas
- Power BI Desktop
- Power Query (M)
- Excel / CSV

## Observações
Os dados são fictícios e utilizados apenas para fins educacionais.
